Different Machine Learning Models

Through completing this assignment, I gained a deeper understanding of how different machine learning algorithms are categorized and applied across various domains. Creating the visual framework helped me clearly distinguish between supervised and unsupervised learning algorithms and recognize how each algorithm is suited for different types of problems. For example, supervised algorithms such as Decision Trees, Random Forests, and Support Vector Machines rely on labeled data for prediction tasks, while unsupervised methods such as K-means clustering and Principal Component Analysis are useful for discovering patterns within unlabeled datasets.

Concepts and study cases

AI Models:

  • Convolutional Neural Network (CNN) for Deep Learning
  • Cox Proportional Hazards Regression combine with PCE

Research Framework:

  • Model Interpretability for Regulatory and Clinical Application
  • Structured vs. Unstructured

Objective

to identify and categorize key machine learning algorithms and understand how they are applied in different domains. I write this to strengthen my ability to communicate technical concepts in machine learning in a structured and accessible way.

Value Proposition

To help myself understand, I create a clear visual framework that organizes key machine learning algorithms by their learning type and application domain. It helps simplify complex AI concepts and demonstrates how different algorithms are used to solve real-world problems across areas such as tabular data analysis, computer vision, natural language processing, and generative AI. By presenting these relationships visually, the framework improves understanding of how algorithms differ and when they should be applied. It also serves as a useful learning and reference tool for understanding the broader machine learning ecosystem.
